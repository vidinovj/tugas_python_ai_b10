# Tugas Python AI B10 - Rangkuman Struktur Data, Fungsi, & Analisis Data

Repository ini berisi kumpulan tugas pemrograman Python untuk program AI B10. Berikut adalah rangkuman dari Tugas 4 hingga Tugas 6.

## Rangkuman Tugas

### Tugas 4: Python Data Structures
Fokus pada penguasaan berbagai struktur data fundamental di Python:
- **List & Tuple**: Manipulasi urutan data dan pemahaman tentang immutability.
- **Set**: Operasi himpunan (union, intersection, dsb) dan sifat unik elemennya.
- **Dictionary**: Manajemen data berbasis key-value.
- **Comprehension**: Teknik penulisan kode yang lebih ringkas (Pythonic) untuk membuat list, dict, dan set.

### Tugas 5: Python Function & Class
Fokus pada konsep modularitas dan Pemrograman Berorientasi Objek (OOP) dasar:
- **Function**: Penggunaan fungsi dengan parameter default dan type hinting.
- **Class & Object**: Implementasi class `Student` untuk mengelola data nilai dan status kelulusan secara terorganisir.
- **Modularitas**: Bagaimana fungsi dapat digunakan kembali di dalam method sebuah class.

### Tugas 6: Python Modules, File I/O, & Analisis Data
Fokus pada penggunaan library eksternal dan manajemen file:
- **NumPy**: Digunakan untuk pengolahan data numerik dan statistik dasar secara cepat.
- **Pandas**: Digunakan untuk manajemen data tabular (DataFrame) dan transformasi data (filtering/kolom baru).
- **File I/O**: Menyimpan hasil analisis dan ringkasan statistik ke dalam file eksternal `.txt`.
- **Advanced OOP**: Integrasi DataFrame ke dalam class `GradeBook` untuk analisis data yang lebih kompleks.

## Kesimpulan
Melalui tugas-tugas ini, pemahaman tentang Python telah berkembang dari sekadar struktur data dasar menuju pengelolaan data yang lebih profesional menggunakan library standar industri (NumPy & Pandas) serta penerapan pola desain OOP untuk skalabilitas kode.
